Ciclo de Execução

O Python não é apenas "lido" pela máquina. O código que você escreve passa por uma fase de compilação rápida para Bytecode, que é uma linguagem intermediária lida pela PVM (Python Virtual Machine).

Código Fonte (.py) 
vazao = 10
tempo = 5
print(vazao * tempo)
Bytecode (.pyc)

São instruções de baixo nível (como LOAD_CONST, BINARY_MULTIPLY) que a máquina virtual do Python entende e executa de forma muito mais rápida que o texto puro.

Rastreio de Código

vazao = 150
tempo = 10
volume = vazao * tempo
print(volume)

Memória Principal (RAM)

Endereço 0x1 [vazao] -
Endereço 0x2 [tempo] -
Endereço 0x3 [volume] -
